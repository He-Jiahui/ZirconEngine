# Editor Shell SVG Resources

The `zircon_editor_shell` icon pack is a repository-owned SVG resource set for the Editor shell visuals shown in the reference screenshot. It does not replace the existing Ionicons subset under `zircon_editor/assets/icons/ionicons`; it provides project-specific glyphs where the screenshot needs a consistent Zircon dark-editor visual language.

## Ownership

- `toolbar/` covers top command-bar actions such as menu, file, folder, save, undo/redo, selection tools, transform tools, play, grid, lighting, and overflow.
- `activity/` covers the left activity rail: play, cube, graph, image, audio, code, settings, and help.
- `scene/` covers hierarchy rows and per-row controls: root cube, lighting, sky, geometry, props, player start, audio zone, visibility, lock, and filter.
- `inspector/` covers property-panel section glyphs and common inspector affordances: transform, mesh renderer, link, checked state, and material swatch.
- `controls/` covers component-lab/control-surface glyphs such as add, delete, checkbox, radio, list, and table.
- `status/` covers info, success, warning, and error alert badges.
- `viewport/` covers viewport overlays and status-bar controls such as axis gizmo, grid, magnet snap, globe/world, and crosshair.

## Asset Contract

All files are plain SVG, self-contained, and use only inline geometry, gradients, strokes, and text where needed for the viewport axis labels. No generated file references fonts, raster images, remote resources, or external symbol sprites.

The pack is intentionally unbound to Editor runtime code in this slice. Future UI surfaces can load these files from the asset catalog or copy the same path names into retained-host template data without changing the resource naming contract.
